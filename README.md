<h1 align="center">Aesthetic Accessories</h1>

This is the main web app for Aesthetic Accessories. It is designed to be responsive and accessible on a range of different screen sizes, making it easy to navigate for potential buyers. 

## User Experience (UX)

-   ### User Stories

    -    #### First Time User Goals 
        1. As a first time user, I want to easily be able to view a list of the products
        2. As a first time user, I want to be able to view a list of the individual product details. 
        3. As a first time user, I want to be able to quickly identify deals and special offers. 
        4. As a first time user, I want to easily be able to register.
        5. As a first time user, I want email confirmation after registering. 
        6. As a first time user, I want to be able to select the quantity of the product I'm purchasing.
     
    -   #### Returning User Goals 
        1. As a returning user, I want to easily be to view my profile
        2. As a returning user, I want to be able to see my past purchases.
        3. As a returning user, I want to be able to login to my profile. 
        4. As a returning user, I want my default delivery information. 
        5. As a returning user, I want to easily update my delivery information.
        6. As a returning user, I want to know my shippiing details.
    
    -   #### Frequent User Goals
        1. As a frequent user, I want to be able to continuously checkout easily.
        2, As a frequesnt user, I want to be able to update my profile. 
        3. As a frequesnt user, I want to be updated when there are changes made. 
    
    -   #### Administrator Goals
        1. As an administrator, I want to be able to be able to add products.
        2. As an administrator, I want to be able to delete products.
        3. As an administrator, I want to be able to edit products.

### Testing User Stories from User Experience (UX) Section
-       First Time User Goals 
        
        1. As a first time, I want to easily be able to view the products. 

                1. When entering the site there is a shop now button that takes you straight to the all accessories page. 
                2. The main navbar has links to the all accessories page and various different ways of sorting of the products. 
        
        2. As a first time user, I want to be able to view a list of the individual product details.

                1. When you click a product it will take you to that product's specific product details where you can then add the product to your cart. 

        3. As a first time user, I want to be able to quickly identify deals and special offers.
                
                1. In the main navbar there is a link to the special offers pages. 
                2. You can also see the specific type of special offer such as clearance. 

        4. As a first time user, I want to easily be able to register.

                1. At the top of the page there is a user icon that when you click it it will allow you to either register or login. 
                2. If you are in the sign in page and need to sign up there is a link to sign up at the bottom of the page. 
        
        5.  As a first time user, I want email confirmation after registering. 
                1. When you sign up there is an email sent to you so that you can verify.

        6. As a first time user, I want to be able to select the quantity of the product I'm purchasing.
                1. On the product details page there is a quantity selector where you can select up to 99 items and checkout. 
                2. In your bag you can update the quantity before checking out. 

-       Returning User Goals 
        
        1. As a returning user, I want to easily be to view my profile
                1. In the main navbar there is a link to your profile. 
        2. As a returning user, I want to be able to see my past purchases.
                1. In the profile page you can see past purchases and when you click on the order number you can see the order details. 
        3. As a returning user, I want to be able to login to my profile.
                1. In the my account icon you can click it to login. 
                2. If you go to the signup page there is a link to the login page. 
        4. As a returning user, I want my default delivery information. 
                1. In your profile page you will see your information from your last checkout as your default delivery information.  
                3. Your default delivery information will be filled in when checking out for more convenience.
        5. As a returning user, I want to easily update my delivery information.
                1. You can update your default delivery information in the profile page.
                2. If you haven't filled it out your delivery information will be filled in.
        6. As a returning user, I want to know my shippiing details.
                1. You can see the details after checking out. 
                2. You can go to your past deliveries to see order details. 

-       Frequent User Goals 
        1. As a frequent user, I want to be able to continuously checkout easily.
                1. The checkout form is quick and easy and your delivery information can be filled in with your profile's default delivery information. 
        2. As a frequesnt user, I want to be able to update my profile.
                1. You can update default delivery details.
                2. Your past orders update everytime you order. 
        3. As a frequesnt user, I want to be updated when there are changes made.
                1. Whenever the updates are made the user can see them. 
        
-       Administrator Goals 
        1. As an administrator, I want to be able to be able to add products.
                1. When you're signed in as an administrator you can go to the product management page and add products.
        2. As an administrator, I want to be able to delete products.
                1. When you're signed in as an administrator you can edit products by going onto them in the product details page. 
                2. When scrolling through the products page there are delete products that when you click you delete the product. 
        3. As an administrator, I want to be able to edit products.
                1. When you're signed in as an administrator when you scroll there are edit buttons that you can click that will take you to the edit product page.
                2. When you're on the product details page there is an edit button that can take you to the edit product page.
